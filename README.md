# Virtual  Lab 

## Project Overview
Virtual Flow Lab is a full-stack system to:
- Book lab slots
- Control hardware (NI MyDAQ)
- Submit feedback
- View analytics dashboard

---

##  Tech Stack
Frontend:
- HTML, CSS, JS

Backend:
- Node.js
- Express.js

Database:
- MySQL

---

##  Features
- User Authentication
- Slot Booking System
- Lab Control Interface
- Feedback System
- Admin Dashboard

---

## ⚙️ Setup Instructions

### Backend
cd backend
npm install
npm start

### Frontend
Open index.html in browser
